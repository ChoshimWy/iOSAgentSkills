# HTML 文档统一模板总览

## 默认风格
- 以 **Notion 风格** 为主：浅色、克制、轻边框、正文优先。
- 如果用户明确要求深色或展示型风格，再切换到深色模板。

## 目标
- 让所有对外 HTML 文档统一为“正式、克制、可分享阅读”的风格。
- 保持信息密度，优先清晰结构而不是装饰性效果。

## 固定骨架
1. Hero
   - 标题
   - 日期 / 版本 / 来源 / 作者 / 仓库（分别独立成行）
   - 范围标签
2. 目录
   - 支持锚点跳转
   - 长文优先带 TOC
3. 正文
   - 分章节、子章节
   - 表格 / 列表 / 代码块清晰分隔
4. 附录
   - 示例 JSON
   - 命令
   - 备注 / 风险 / 验收
5. 页脚
   - 文件路径
   - 生成说明

## 统一视觉原则
- 白底或极浅灰底
- 主内容区宽度适中，优先单栏阅读
- 表格表头浅灰，长文本自动换行
- 代码块使用浅灰底等宽字体
- 风险 / 提示 / 结论使用轻 callout 区块
- Hero 元信息逐行排布，不要把日期、来源、作者塞在同一行

## 默认适用文档
- 方案文档
- 任务清单
- 接口说明
- 评审稿
- 运行报告
