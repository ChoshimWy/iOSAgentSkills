# 方案 / 提案 HTML 模板

## 默认风格
- Notion 风格优先：浅色、轻边框、正文优先、目录清晰。

## 结构
- Hero：标题、日期、来源、适用范围、状态
- 目录：自动锚点跳转
- 背景与问题
- 目标与原则
- 方案总览
- 详细设计
- 接口 / 数据模型 / 时序
- 风险与回退
- 工时 / 里程碑（如适用）
- 总结

## 必须包含
- 结论先行
- 问题与目标明确区分
- 架构 / 数据 / 同步 / 播放 / UI 边界说明
- 风险与回退路径
- 需要时给出接口请求 / 返回示例

## 示例区块建议
- 用 callout 表示“核心结论”
- 用 table 表示“模块拆分 / 对比 / 风险矩阵”
- 用 pre/code 表示“JSON / 命令 / 配置”
