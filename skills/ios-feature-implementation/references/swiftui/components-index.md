# 组件索引

这份索引用于快速定位最合适的参考文档。

## 组件入口
- `references/swiftui/tabview.md`：多标签应用和标签驱动功能。
- `references/swiftui/navigationstack.md`：push 导航、路由状态和深链跳转。
- `references/swiftui/sheets.md`：模态展示与集中式 sheet 路由。
- `references/swiftui/form.md`：设置页、分组输入和结构化表单。
- `references/swiftui/list.md`：列表、分组 section 和 feed 样式内容。
- `references/swiftui/scrollview.md`：自定义滚动布局、横向滚动和懒加载栈。
- `references/swiftui/grids.md`：网格、媒体墙和图标选择器。
- `references/swiftui/searchable.md`：搜索栏、scope 和异步搜索结果。
- `references/swiftui/loading-placeholders.md`：骨架屏、占位态和空态。
- `references/swiftui/controls.md`：`Toggle`、`Picker`、`Slider` 等控件。
- `references/swiftui/input-toolbar.md`：底部输入条和聊天编辑器。
- `references/swiftui/top-bar.md`：顶部选择条、胶囊栏和 `safeAreaBar(.top)`。
- `references/swiftui/overlay.md`：toast、banner、临时浮层。
- `references/swiftui/media.md`：远程媒体加载和全屏查看。
- `references/swiftui/matched-transitions.md`：平滑过渡和 source/destination 动画。
- `references/swiftui/title-menus.md`：导航标题菜单。
- `references/swiftui/menu-bar.md`：macOS / iPadOS 命令菜单。
- `references/swiftui/macos-settings.md`：macOS `Settings` 窗口。
